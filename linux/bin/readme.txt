Sample LINUX server starting script.

Prerequisities:
- latest z_srv_pam_codextended.pk3
- put libcod1.so into Call of Duty server directory
- put server_config_1.cfg into main/ Call of Duty
- update startmyserver.sh script content to your needs:
	- HOME variable with your Call of Duty directory
	- +exec server_config_1.cfg replace 'server_config_1.cfg' with your server config filename if needed

1. Place z_srv_pam_codextended.pk3 in main/ of your Call of Duty server directory.
2. Run startmyserver.sh to start Call of Duty server.
